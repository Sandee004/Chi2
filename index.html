Chi-Square (χ²) Distribution Critical Values Table
Critical Values for Various Significance Levels
How to Use This Table:
1. Calculate degrees of freedom: df = (rows - 1) × (columns - 1)
2. Choose your significance level (α)
3. Find the intersection of your df row and α column
4. Compare your calculated χ² with the critical value:
   • If χ²calculated > χ²critical: Reject H₀ (significant)
   • If χ²calculated ≤ χ²critical: Fail to reject H₀ (not significant)

Key Formulas:
• Degrees of Freedom: df = (r - 1)(c - 1), where r = rows, c = columns
• Chi-Square Test Statistic: χ² = Σ[(Observed - Expected)²/Expected]
• Expected Frequency: E = (Row Total × Column Total)/Grand Total
Example for Your Problem (χ²cal = 11.08):
• If df = 2 and α = 0.05: χ²critical = 5.991
• Since 11.08 > 5.991, reject H₀ (significant association)
• If df = 3 and α = 0.01: χ²critical = 11.345
• Since 11.08 < 11.345, fail to reject H₀ (not significant at 1% level)
